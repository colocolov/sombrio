Gulp-команды шаблона

Запуск в режиме разработки: gulp (изображения не оптимизируются, svg-иконки объединяются в один спрайт-файл, шрифты не конвертируются)

Сборка оптимизированной версии для продакшена: gulp build

Запуск конвертации шрифтов: gulp fonts (запускать отдельно, если нужно сконвертировать шрифт)

Запуск создания svg-spirite: gulp svg

Запуск создания favicon: gulp favicon (иконку добавить в /images/favicon/ и запустить таск)

Примеры svgSprite

Иконки размещаем в папку: src/images/icons

Готовый файл со спрайтом собирается в: папка_проекта/images/icons/sprite.svg

   

HTML код

<svg class="icon-svg">
   <use href="images/icons/sprite.svg#instagram"></use>
 </svg>

CSS код

.icon-svg {
   fill: transparent;
   stroke: transparent;
   width: 40px
   height: 40px
   transition: fill 0.3s ease-in
}

Горячие клавиши и миксины

@mixins

To use: in sass - @include+hotkeys
  • image-shadow => shadow around img
  • hide => hide block/element

--adaptiv media sass

To use: in sass - hotkeys
  • desktop => desktop (1129.98px)
  • pc => notebook (1023.98p)
  • tablet => tablet (991.98px)
  • mob => phone (767.98px)
  • mobMid => phone mid (575.98px)
  • mobSm => phone small (479.98px)
  • mma => media max
  • mmi => media min

--html

To use: in html - hotkeys
  • picture => tag picture with srcset
  • img => tag img in wrap with class _ibg
  • img2x => tag img for 2xRETINA in wrap
  • swiper => slider on swiper
  • menusoc => socials menu
  • check => checkbox
  • sec => section with class and container
  • d => tag div with class
  • svg => include svg with hover
  • vbg => video on background
  • hide => hide block

--sass

To use: in sass - hotkeys
  • adap => adaptive property
  • fo => include font: weight / size / line-height
  • psaimg => position-absolute-img
  • trs => transition

--hotkeys VSCode

To use: in VSCode - hotkeys
  • Shift + ` => ECSSTractor
  • Alt + D => обернуть выделенное в тег
  • Ctrl + G => переход к нужной строке
  • Ctrl + P => переход к файлу
  • Alt + Shift + F => форматирование кода
  • F2 => переименование переменной в JS

Code

Код для изменения текста в браузере (вставить в консоль)
  • document.body.contentEditable=true
